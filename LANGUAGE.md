## Source Code

- The source code is maintained in its original form.
- All code comments are written in **English**.
- Identifiers (variables, functions, classes) remain unchanged unless technically required.

## Documentation

- The primary documentation language is **English**.
- Optional additional documentation may be provided in another language.

## Rationale

English is used to ensure:
- readability for the team,
- long-term maintainability,
- compatibility with common open-source tooling.

## Policy Application

This policy applies to all newly added or modified code.
